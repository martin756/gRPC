Ejecutar en el siguiente orden dentro de la carpeta grpc_server con consola cmd:
1. Comprobar tener instalado con: python -V
2. pip install pipenv
3. pipenv shell
4. pipenv install grpcio grpcio-tools googleapis-common-protos mysql-connector-python
5. python -m grpc_tools.protoc -I..\gRPC_server\protobufs --python_out=. --grpc_python_out=. ..\gRPC_server\protobufs\usuarios.proto -> Generar codigo para cada uno de los archivos protos
5.1. python -m grpc_tools.protoc -I..\gRPC_server\protobufs --python_out=. --grpc_python_out=. ..\gRPC_server\protobufs\productos.proto
5.2. python -m grpc_tools.protoc -I..\gRPC_server\protobufs --python_out=. --grpc_python_out=. ..\gRPC_server\protobufs\carritos.proto
7. python .\server.py -> Compilar y ejecutar servidor
